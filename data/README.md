# Memorial File

The display reads daily memorials from one of these files, in this order:

1. `memorials.xlsx`
2. `memorials.xls`
3. `memorials.csv`

Recommended columns:

- `name`
- `day`
- `month`
- `year`
- `afterSunset`

You can also use Hebrew headers:

- `שם`
- `יום`
- `חודש`
- `שנה`
- `אחרי שקיעה`

Example row:

```text
דוגמה בן דוגמה,כט,אדר,תשפא,false
```
